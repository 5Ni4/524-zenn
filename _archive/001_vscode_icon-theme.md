# VScodeでディレクトリをアイコンとして表示したい！

## きっかけ

先日、久しぶりにPCを新調した。
業務端末のセットアップであれば、去年10月に実施したが記憶が全然ない。

本格的にVScodeを触って作業していこうと思ったところ、早速ディレクトリにフォルダアイコンが付いていないのを発見した（`画像1`）。

![画像1: フォルダアイコンのないVScode](_draft_images/image-1.png)

## 手順

これって何か特別な設定いったっけ？とか一瞬思ったけど、普通に **file icon theme を適用**するだけでオールクリア。

1. 左下の歯車⚙️アイコンをクリック
2. テーマ > ファイルアイコンのテーマ をクリック（`画像2`）

![画像2: ファイルアイコンのテーマ設定の入り口見つける](<_draft_images/スクリーンショット 2025-11-02 20.30.35.png>)

3. 開いたメニューの一番下に「その他のファイル アイコンテーマをインストール」があるのでクリック（`画像3` ※ 筆者はインストール済みだったので色々出てしまっている）

![ファイルアイコンテーマのインストールへの導線](_draft_images/image-3.png)

4. 拡張機能の検索ページから、気になったアイコンテーマをインストール

![画像3: 拡張機能ページ](_draft_images/image-2.png)

5. 適用される！

![画像4: ファイルアイコンついた！](_draft_images/image.png)

## 備考

今回適用しているのは 「[Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme)」。

意味をなす命名をするとディレクトリのアイコンがいい感じになってくれるのが良い。ちょっとしたモチベ維持にもなるのでおすすめ。
