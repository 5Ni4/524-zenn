## なぜこのフォルダがあるのか

### スラッグがランダム文字列

Zennの記事投稿を初めてすぐの時、**スラッグ指定を見落としてて**ぐちゃぐちゃの文字列になってしまった＼(^o^)／

Git連携では**ファイル名 = スラッグ**なので他記事といっしょにしてしまうと管理がつらい

### フォルダの役割

- **全記事をリポジトリに揃えたい**（001, 002が欠けてるのが気持ち悪い）
- ローカルでも全記事を一箇所で管理したい
- 記事の変遷を記録として残したい

※ リライトはZennのブラウザ上でやる
